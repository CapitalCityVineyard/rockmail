Installation

To edit the default emails for all new event registration templates, modify the registration template details block.

  1. Open Event Registration Page
  2. Select Add Registration Template
  3. Use the administration bar to open Block Configuration
  4. Open Block Properties on the Registration Template Detail Block
  5. Paste Compiled HTML (From dist folder)
